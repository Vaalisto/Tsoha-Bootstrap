## Työn aihe

Työn aiheena on lautapeleihin liittyvä websovellus, johon voidaan tallentaa erilaisia lautapelejä ja määritellä niille genrejä. Jos aikaa riittää, niin pelejä voidaan myös kommentoida ja arvostella.
